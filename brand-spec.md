# Conex-Iones — brand-spec

Extracted from project brief (mandatory visual direction).

## Color tokens (OKLch + source hex)

```css
:root {
  --bg:      oklch(14% 0.014 250);   /* #0b1117 */
  --surface: oklch(18% 0.018 245);   /* #111a22 */
  --fg:      oklch(96% 0.006 240);   /* near-white */
  --muted:   oklch(68% 0.02 240);    /* soft blue-grey */
  --border:  oklch(28% 0.02 245);    /* hairline on dark */
  --accent:  oklch(78% 0.12 175);    /* #2dd4bf turquoise */
}
```

## Typography

- **Display:** Outfit, system-ui, sans-serif (strong, clean)
- **Body:** "Source Sans 3", system-ui, sans-serif
- **Mono:** ui-monospace (meta labels only)

## Layout posture

1. Dark elegant canvas — no purple gradients, no glow wash
2. Button radius 8–10px (never pills)
3. One accent used sparingly (CTAs, active links, emphasis)
4. No cards / floating badges in hero
5. One purpose + one title + one short lead per section
